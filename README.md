# CTC-Report
本仓库为苏州大学和阿里巴巴联合团队参加[CTC-2021文本纠错比赛](https://github.com/destwang/CTC2021)的评测报告发布页，我们的系统在该评测中分别获得了**查错、改错和总分三个赛道的第一**。具体评测报告参见[此处](https://github.com/HillZhang1999/CTC-Report/blob/main/Report.pdf)。
